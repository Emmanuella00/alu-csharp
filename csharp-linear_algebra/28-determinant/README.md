# 28 - Matrix Determinant

This task implements a method that calculates the determinant of a 2x2 or 3x3 matrix. The 2x2 case uses the closed-form `ad - bc` formula, and the 3x3 case uses cofactor expansion along the first row. Any unsupported dimension returns -1.

## Requirements

- All C# files compiled using `dotnet`
- Projects built with .NET 7 SDK
