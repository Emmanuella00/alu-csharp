# C# Test Driven Development

## Description
This project introduces Test Driven Development (TDD) in C# using NUnit.
Each task contains a class library solving a problem and a test library
validating the solution. Tests are written before the implementation code.

## Technologies
- C# / .NET 7
- NUnit 3.12.0
- NUnit3TestAdapter 3.17.0
- Microsoft.NET.Test.Sdk 16.11.0
